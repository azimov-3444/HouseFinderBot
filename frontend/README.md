# House Finder Frontend

React/Vite frontend for the House Finder real estate platform.

## Setup

```bash
npm install
copy .env.example .env
npm run dev
```

Default URL:

```text
http://localhost:5173
```

## Environment

```env
VITE_API_URL=http://localhost:5000/api
```

Use `http://localhost:5173` during local development so cookie auth matches the backend `CLIENT_URL`.

## Scripts

- `npm run dev` - start Vite
- `npm run build` - production build
- `npm run lint` - run ESLint
- `npm run preview` - preview built app
